File Input and Output, Using Loops to Process Files, Processing Records, Exceptions

Input and Output

read from a file, write from a file. Data is stored as variables and saved to a file for use later.
Files are saved from RAM and stored onto physical storage.
3 steps when a program uses a file. it Open the file, Process the file, and Closes the file

Types of Files and File Access Methods

Text Files - contains data has been encoded as text. uses .txt file
Binary Files - contains data that has not been converted to text.

Two ways to access data stored in a file
Sequential Access - files are read sequentially from beginning to end, NO skipping ahead
Direct Access - can jump directly to any piece of data in the file

File Names and File Objects

we will be using .txt (text) files.
File Object -- object associated with a specific file the object is referenced by a variable

Open Function - open()

creates a file object and associates it with a file on the disk
ex: file_object = open(filename, mode)

Mode - the mode is a string which specifies how to open the file, 'r' = read; 'w' = write; and 'a' = append
you have to specify a path, otherwise the program will automatically create a file with that name in the current directory

Methods are functions which belong to objects

write method which is used to write data to the file ex: file_variable.write(string)

close method is used to close the file object after manipulating the file. ex: file_variable.close()

Reading Data from a File

read method - file must be opened first, this method will read the entire file in RAM and return the contents as a String

readline method - a method to read a line from the file. Lines are returned as a string, including '\n'
Read Position - marks the location of the next item to be read from a file

example: read_print.py

Using Loops to Process Files ex:

    hobbyFile = open(r'c:\Files\myhobbies.txt', 'r')
    hobbyName = hobbyFile.readline()     # priming read
    while hobbyName != '':             # empty string means End of File (EOF)
        hobbyName = hobbyName.rstrip('\n')
        hobbyYears = hobbyFile.readline()
        hobbyYears = int(hobbyYears.rstrip('\n'))
        print(hobbyName, hobbyYears)
        hobbyName = hobbyFile.readline()   #last line of the loop is another read - when the file is done the loop will get an empty string and will move to close
    hobbyFile.close()

using 'for loop' to read contents

        hobbyFile = open(r'c:\Files\myhobbies.txt', 'r')
        for hobbyLine in hobbyFile:
            print(hobbyLine)
        hobbyFile.close()
the above code handles all lines generically; to use them as different fields you need additional code
