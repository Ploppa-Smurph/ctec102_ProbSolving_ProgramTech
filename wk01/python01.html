Compilers and Interpreters

Compilers - take all the code and compile into an executable file which runs on the system. - with compilers you don't have to share your source code, so it is more protected. you simply share the executable.
syntax errors only are exposed when you compile, cannot successfully compile until any syntax error is fixed.

Interpreters - take code from a source, convert the code to binary, and execute the resulting code. with interpreters, syntax errors are revealed when the code is interpreted and will only run when corrected.
